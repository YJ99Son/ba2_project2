# Business Analytics II - Project II

## Overview
This project involves building classification models to predict whether an Airbnb host is a "Superhost".

## Files
- `processing.ipynb`: Jupyter Notebook containing the data analysis, model building, and prediction pipeline.
- `train_data.csv`: Training dataset (filtered columns).
- `test_f25.xlsx`: Test dataset for final prediction.

## Usage
1.  Install dependencies: `pip install pandas numpy matplotlib seaborn scikit-learn openpyxl`
2.  Run `processing.ipynb` to generate the model and predictions.
